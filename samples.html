Profile Page Class Project #1

(HTML, CSS, Bootstap Framework)

This is the first responsive web design project with proper use case that the participants will code with the guidance of our instructor. The participants will be making their own profile pages where they can share more about themselves and the web projects that they have built.

This project aims to recap the fundamentals of HTML, CSS and Bootstrap framework that has been taught to them.

BMI Calculator Class Project #2

(HTML, CSS, Bootstap Framework, Javascript)

This is first functional web app that the participants will code with the guidance of our instructor. The participants will be developing a functional BMI calculator web app that informs its users of their health status according to their BMI results.

This project aims to allow the participants to apply Javascript to make the web app respond to user input appropriately. It is also meant to be a quick recap of HTML, CSS and Bootstrap framework.
